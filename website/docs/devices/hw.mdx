# Supported Peripherals

## Simple analog services

The following can be attached to any analog pin, and have [common configuration](https://microsoft.github.io/devicescript/api/servers/analog).

<!-- ANALOG-START -->
* [LightLevel](https://microsoft.github.io/devicescript/api/clients/lightlevel)
* [ReflectedLight](https://microsoft.github.io/devicescript/api/clients/reflectedlight)
* [WaterLevel](https://microsoft.github.io/devicescript/api/clients/waterlevel)
* [SoundLevel](https://microsoft.github.io/devicescript/api/clients/soundlevel)
* [SoilMoisture](https://microsoft.github.io/devicescript/api/clients/soilmoisture)
* [Potentiometer](https://microsoft.github.io/devicescript/api/clients/potentiometer)
<!-- ANALOG-END -->

## HID Services

These are currently only supported on RP2040. There is no configuration. They emulate HID devices.

<!-- HID-START -->
* HID Mouse
* HID Keyboard
* HID Joystick
<!-- HID-END -->

## Other services

These require specific configuration.

<!-- SERVERS-START -->
* [RotaryEncoder](https://microsoft.github.io/devicescript/api/servers/rotaryencoder)
* [Button](https://microsoft.github.io/devicescript/api/servers/button)
* [Relay](https://microsoft.github.io/devicescript/api/servers/relay)
* [Power](https://microsoft.github.io/devicescript/api/servers/power)
* [Accelerometer](https://microsoft.github.io/devicescript/api/servers/accelerometer)
<!-- SERVERS-END -->
