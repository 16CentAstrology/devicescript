# MSR JM Brain S2-mini 207 v4.2




- Jacdac on pin 17 using Jacdac connector
- RGB LED on pins 8, 7, 6
- Service: power




## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash esp32 --board msr207_v42
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32s2-msr207_v42-0x1000.bin)
