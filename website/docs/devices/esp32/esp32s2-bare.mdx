# Espressif ESP32-S2 (bare)

A bare ESP32-S2 board without any pin functions.


- [Store](https://www.espressif.com/en/products/socs/esp32-s2)




## Firmware update

In [Visual Studio Code](/getting-started/vscode),
select **DeviceScript: Flash Firmware...** from the command palette.
        
Run this [command line](/api/cli) command and follow the instructions.

```bash
devicescript flash esp32 --board esp32s2_bare
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32s2-esp32s2_bare-0x1000.bin)
