# Espressif ESP32 (bare) DeviceScript

Bare ESP32 without any default functions for pins.


- [Store](https://www.espressif.com/en/products/socs/esp32)




## Firmware update

```bash
devicescript flash esp32 --board esp32_bare
```


- [Firmware](https://github.com/microsoft/jacdac-esp32/releases/latest/download/devicescript-esp32-esp32_bare-0x1000.bin)
