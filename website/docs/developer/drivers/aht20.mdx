# AHT20

Driver for AHT20 temperature/humidity sensor at [I2C](/developer/i2c/) address `0x38`.

-   Services: [temperature](/clients/api/temperature/), [humidity](/clients/api/humidity/)
-   [Datasheet](https://asairsensors.com/wp-content/uploads/2021/09/Data-Sheet-AHT20-Humidity-and-Temperature-Sensor-ASAIR-V1.0.03.pdf)
-   [Source](https://github.com/microsoft/devicescript/blob/main/packages/drivers/src/aht20.ts)

## Usage

```ts
// highlight-next-line
import { startAHT20 } from "@devicescript/drivers"

// highlight-next-line
await startAHT20()
const t = new ds.Temperature()
const h = new ds.Humidity()
```
